# Better Stack (better-stack)
Better Stack is a comprehensive infrastructure monitoring platform that helps teams identify and resolve website and server issues quickly by combining features like uptime monitoring, log management, incident management, and status pages, essentially alerting users when their website goes down and providing tools to diagnose the problem with detailed error logs and screenshots; allowing them to minimize downtime and improve user experience.
